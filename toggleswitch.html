ToggleSwitch

The ToggleSwitch component is used as an alternative for the Checkbox component, switch between enabled and disabled states.

Import

import ToggleSwitch from @trendmicro/react-styled-ui/ToggleSwitch";
// or
import { ToggleSwitch } from @trendmicro/react-styled-ui";

Usage

<Flex align="center">
  <ToggleSwitch pr="1x" id="email-alerts" />
  <label htmlFor="email-alerts">Enable email alerts?</label>
</Flex>
EDITABLE EXAMPLE

Sizes

Pass the size prop to change the size of the ToggleSwitch.

<Flex align="center">
  <ToggleSwitch size="sm" />
  <ToggleSwitch size="md" />
  <ToggleSwitch size="lg" />
</Flex>
EDITABLE EXAMPLE

ToggleSwitch background color

  • Use the variantColor prop to change the checked background color of the ToggleSwitch. variantColor can be any color key with key 50(hover), 60(checked, focus) that exist in the theme.colors.

    That means the red:60 exists in the theme.colors then can use variantColor="red".

  • Learn more about colors theming.

<Flex align="center">
  <ToggleSwitch defaultChecked variantColor="red" />
  <ToggleSwitch defaultChecked size="lg" variantColor="green" />
</Flex>

Props

NameTypeDefaultDescription
sizesm, md, lgmdThe size of the switch.
variantColorstringblueThe background color of the switch when checked.
namestringThe input name of the switch when used in a form.
valuestring, booleanThe value of the switch.
childrenReact.ReactNodeThe children of the switch.
aria-labelstringThe aria-label of the switch for accessibility.
aria-labelledbystringThe aria-labelledby of the switch for accessibility.
checkedbooleanIf true, set the switch to the checked state.
defaultCheckedbooleanIf true, the checkbox will be initially checked.
